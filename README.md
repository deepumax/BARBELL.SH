# BARBELL.SH

BARBELL.SH is a modern fitness and gym website designed with a unique developer-inspired aesthetic. The project features responsive layouts, membership plans, training sections, booking forms, facility showcases, and a clean high-performance UI focused on strength training and fitness culture.

## Features

- Responsive modern UI
- Gym membership plans
- Trainer and class booking section
- Facility showcase
- Developer-themed branding
- Smooth layout and clean design
- Mobile-friendly experience

## Tech Stack

- HTML
- CSS
- JavaScript
